Este proyecto lo desarrollamos como grupo como una forma de mostrar nuestra creatividad y lo que hemos aprendido trabajando juntos. Decidimos crear un quiz interactivo sobre tecnología porque refleja nuestros intereses y nos permite compartir conocimientos de una manera entretenida y sencilla. Más allá del código, este trabajo representa nuestro esfuerzo personal, nuestra colaboración y el compromiso de aplicar lo aprendido en un producto que nos identifica y que demuestra que podemos trabajar en equipo para lograr un resultado propio y significativo.
